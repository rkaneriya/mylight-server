## Setup 
1) Install PostgreSQL via Homebrew

```
npm run postgres
```

2) Run PostgreSQL DB Server

```
postgres -D /usr/local/var/postgres
```

3) Install dependencies 

```
npm install 
```

4) Run application

```
npm start (to start Express server)
```
